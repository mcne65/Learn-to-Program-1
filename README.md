# Learn-to-Program
A public repository containing a large list of PDFs and resources I've compiled

## C++
### Beginners
* [Programming Principles and Practice Using C++ (by Bjarne Stroustrup, highly recommended)](https://ebooks-it.org/0321992784-ebook.htm)
* [Thinking in C++](http://iacs-courses.seas.harvard.edu/courses/cs207/resources/TIC2Vone.pdf)
* [Accelerated C++](http://irpp.org/wp-content/uploads/sites/2/2014/04/Andrew-Koening-and-Barbara-E.-Moo-Accelerated-C++.pdf)
* [C++ Primer](http://www4.zippyshare.com/v/9R2PZu3Z/file.html)

### Practices
* [Effective C++](https://www.epiportal.com/Ebooks/Addison.Wesley.Effective.CPP.3rd.Edition.May.2005.pdf)
* [Effective Modern C++](https://disciplinas.stoa.usp.br/pluginfile.php/1995323/mod_resource/content/1/Effective%20Modern%20C%2B%2B%202014.pdf)
* [Effective STL](http://d1.amobbs.com/bbs_upload782111/files_11/ourdev_482463.pdf)
